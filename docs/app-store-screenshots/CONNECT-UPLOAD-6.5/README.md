# Скриншоты для App Store Connect — iPhone 6.5" Display

**Перетащите эти 4 файла в App Store Connect → iPhone 6.5" Display**

## Файлы (порядок загрузки)

| № | Файл | Размер |
|---|------|--------|
| 1 | `01-diagnostic.png` | 1284×2778 px |
| 2 | `02-parts.png` | 1284×2778 px |
| 3 | `03-locator.png` | 1284×2778 px |
| 4 | `04-results.png` | 1284×2778 px |

Размер **1284×2778** указан в требованиях Apple для iPhone 6.5" (наряду с 1242×2688). Отдельные варианты 1242×2688 не нужны.

## Как загрузить

1. Откройте [App Store Connect](https://appstoreconnect.apple.com) → ваше приложение.
2. **App Store** → версия (например 1.0) → **Previews and Screenshots**.
3. Выберите вкладку **iPhone 6.5" Display**.
4. Перетащите все 4 PNG из этой папки в область загрузки (или нажмите «Choose File»).
5. Убедитесь, что порядок: Diagnostic → Parts → Locator → Results.
6. Нажмите **Save** вверху страницы.
