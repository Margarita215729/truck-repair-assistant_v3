# Скриншоты для App Store Connect — iPhone 6.5" Display

**Перетащите эти 4 файла в App Store Connect → iPhone 6.5" Display**

## Формат (исправлено)

| Параметр | Значение |
|----------|----------|
| Формат файла | **PNG** (не JPEG с расширением .png) |
| Цвет | sRGB, 8-bit RGB |
| Альфа-канал | нет |
| Размер | **1242×2688** px (portrait) |

Предыдущая версия была отклонена: `sips` сохранил JPEG-данные в файлы с расширением `.png`. App Store Connect проверяет реальный формат, а не расширение.

## Файлы (порядок загрузки)

| № | Файл | Размер |
|---|------|--------|
| 1 | `01-diagnostic.png` | 1242×2688 px |
| 2 | `02-parts.png` | 1242×2688 px |
| 3 | `03-locator.png` | 1242×2688 px |
| 4 | `04-results.png` | 1242×2688 px |

Альтернативный принятый размер для 6.5": 1284×2778. Используйте **один** из двух наборов, не оба.

## Как загрузить

1. Откройте [App Store Connect](https://appstoreconnect.apple.com) → ваше приложение.
2. **App Store** → версия (например 1.0) → **Previews and Screenshots**.
3. Выберите вкладку **iPhone 6.5" Display**.
4. Перетащите все 4 PNG из этой папки в область загрузки.
5. Порядок: Diagnostic → Parts → Locator → Results.
6. Нажмите **Save**.

> **Рекомендация:** если доступна вкладка **iPhone 6.9" Display**, загрузите скриншоты из `CONNECT-UPLOAD-6.9/` — Apple использует их как основной размер и масштабирует для меньших устройств. Вкладка 6.5" нужна только если 6.9" не загружены.
