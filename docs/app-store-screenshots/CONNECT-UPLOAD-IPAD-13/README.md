## Загрузка в App Store Connect

1. **App Store** → версия **1.0**
2. **Previews and Screenshots** → вкладка **iPad**
3. Секция **iPad Pro 13" Display** (или **13" Display**)
4. Перетащите все 4 PNG из этой папки
5. **Save**
