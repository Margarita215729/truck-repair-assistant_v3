# Иконка для App Store Connect — App Information

**Загрузите `AppIcon-1024.png` в App Store Connect → App Information → App Icon**

| Файл | Размер |
|------|--------|
| `AppIcon-1024.png` | 1024×1024 px, PNG, без прозрачности |

## Как загрузить

1. App Store Connect → ваше приложение → **App Information** (слева).
2. В блоке **App Icon** нажмите **Choose File** или перетащите файл.
3. Сохраните изменения.
