# Скриншоты для App Store Connect — iPhone 6.9" Display

**Перетащите эти 4 файла в App Store Connect → iPhone 6.9" Display**

## Формат

| Параметр | Значение |
|----------|----------|
| Формат файла | PNG |
| Цвет | sRGB, 8-bit RGB |
| Альфа-канал | нет |
| Размер | **1290×2796** px (portrait) |

Другие принятые размеры для 6.9": 1320×2868, 1260×2736. Используйте один набор.

## Файлы (порядок загрузки)

| № | Файл | Размер |

## Как загрузить

1. Откройте [App Store Connect](https://appstoreconnect.apple.com) → ваше приложение.
2. **App Store** → версия → **Previews and Screenshots**.
3. Выберите вкладку **iPhone 6.9" Display** (основной размер для iPhone в 2025–2026).
4. Перетащите все 4 PNG из этой папки.
5. Порядок: Diagnostic → Parts → Locator → Results.
6. Нажмите **Save**.

Если загружены скриншоты 6.9", вкладка **iPhone 6.5" Display** не обязательна — Apple масштабирует автоматически.
