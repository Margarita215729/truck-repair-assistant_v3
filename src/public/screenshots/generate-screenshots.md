# PWA Screenshots Generation

For mobile app stores, you need screenshots in specific formats:

## Required Screenshot Sizes:

### Mobile (Portrait)
- 375x812 (iPhone X/11/12/13 mini)
- 414x896 (iPhone 11/XR)
- 390x844 (iPhone 12/13)
- 393x852 (iPhone 14)

### Tablet (Landscape)
- 1024x768 (iPad)
- 1366x1024 (iPad Pro)

## How to Generate:

1. **Manual Screenshots:**
   - Open app in device simulator
   - Take screenshots of key features
   - Resize to exact dimensions

2. **Automated Screenshots:**
   - Use Playwright or Puppeteer
   - Set viewport to target dimensions
   - Capture key screens programmatically

3. **Design Tools:**
   - Create mockups in Figma/Sketch
   - Export in required sizes
   - Add device frames if needed

## Key Screens to Capture:

1. **Main diagnostic screen** (with form)
2. **Analysis results screen** (with AI recommendations)
3. **Repair shops screen** (showing local services)
4. **Parts & costs screen** (showing estimates)
5. **Tow trucks screen** (emergency services)

## Notes:
- Use real data in screenshots
- Show app's key value propositions
- Ensure screenshots are high quality
- Consider light/dark mode variations
